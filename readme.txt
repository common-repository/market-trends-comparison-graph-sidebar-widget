== Description ==

Compare markets in a side by side contest where Google's search statistics show which one is more popular. 
Not many people know that Google tracks what people are searching for and the volume of those searches. 
It is easy to compare markets in a visual way with this graph generator and see which is more popular without resorting to votes. 
You will be pretty shocked at which is more popular then the other, besides being thoroughly entertained. 
This tool will be something your social audience will appreciate using and playing around with. 
Install it and see your web site's interaction sky rocket!

== Installation ==

1. Upload `wp-lmt.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Widgets (under Appearance) and drag "WP Market Trends Comparison Graph" to the sidebar
4. Open "Market Trends Comparison Graph Widget" in sidebar and fill in the title, the first market's name and the second market's name.

5. Save and close sidebar widget.
